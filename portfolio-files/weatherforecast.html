Project Overview

Weather forecast is a simple web-application that includes Open Weather API to display daily weather forecast around the world. Click here to see it.

Tools and Softwares

  • Programming Language: Javascript, HTML, CSS, MySQL
  • Software: Visual Studio Code
  • Libraries: Font-awesome w3schools, Ajax/jQuery

Features

Users will be brought to the first page where they can type any city in the world, then click the button.

The weather information of the searched city will be displayed afterwards.
