Course Selection System

Back-end: PHP Front-end: HTML, CSS, JavaScript, Ajax/jQuery

Project Overview

Course Selection System is a web-based application for students to select courses that they want to enter. There are 2 roles in this website, which are administrator and students.

Tools and Softwares

  • Programming Language: HTML/CSS, PHP, JavaScript, MySQL
  • Software: Visual Studio Code
  • Libraries: Font-awesome w3schools, Ajax/jQuery

Features

Course selection system is created by considering the UI for users so that it doesn't look boring. User will be directed to the login page, where they can either login or register by clicking buttons provided.

Task Specification

1. User Management

The user of this website is categorized into 2 types:

  • Administrator

    • Administrator is the master of this website, which makes this user has the highest authority. Administrator can manage students and courses database.

  • Student

    • Student can log in to the website after registering their account to the website. They are required to register their id, name, and major that they have taken.
