Mantis Bug Tracker

Back-end: PHP Front-end: HTML, CSS, JavaScript, Ajax/jQuery

Project Overview

This website is an open-source bug tracker that is used for project listing and bug reporting. This project is the first open-source project that I contributed.

Tools and Softwares

  • Programming Language: Javascript, HTML, CSS, MySQL
  • Software: Visual Studio Code
  • Libraries: Ajax/jQuery

Features

Mantis is an open-source bug tracker that is used for project listing and bug reporting. The use of this bug tracker is similar to Jira Software. There are two roles in this website, which is administrator and reporter.

Search and Filter

There are some things that I have changed, such as this search field that I put in the home page for administrators to search and filter for the issues based on reporter or status.

Simpler Issue Status Change

Usually, users has to open one issue to edit its status. But after modification, this website can now edit issue without opening the issue.

When user opens a bug list page, they can also edit the status, priority, severity, and even the issue summary.

Image Preview

When reporter uploads some images about reported bugs, user can now preview the image without have to be redirected to another page. By simply click the image, the image will be zoom in and user can navigate through pictures shown in carousel style.
